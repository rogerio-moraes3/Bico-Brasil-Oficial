# Bico Brasil — Documento Mestre

## O que é o projeto Bico Brasil
O Bico Brasil é uma plataforma web/mobile que conecta pessoas que precisam gerar renda rapidamente com oportunidades de trabalho informal imediato (“bicos”) na sua cidade. O sistema é orientado a acesso direto ao trabalho, com fluxo simples e rápido, evitando intermediações complexas.

## Problema social e econômico que resolve
Pessoas desempregadas ou subempregadas precisam gerar renda diária ou imediata para suprir necessidades básicas (alimentação, contas, transporte). Essas pessoas não encontram soluções rápidas, acessíveis e confiáveis nos canais tradicionais, o que gera lacunas de renda e vulnerabilidade financeira.

## Usuário real do sistema
- Adultos de baixa a média renda, frequentemente desempregados ou em trabalhos informais.
- Escolaridade variada (fundamental a médio).
- Acesso predominantemente via celular, com internet móvel limitada.
- Baixo tempo disponível e baixa tolerância a processos longos ou confusos.
- Necessidade de urgência financeira, com foco em renda no curto prazo.

## Objetivo principal do sistema
Permitir que o usuário encontre e execute um bico de forma rápida, clara e prática, gerando renda no menor tempo possível.

## O que o sistema explicitamente NÃO é
- Não é rede social.
- Não é marketplace complexo.
- Não é plataforma corporativa de RH.
- Não é app de freelancing profissional.
- Não é sistema de anúncios genéricos sem curadoria mínima.

## Princípios fundamentais do produto
- **Simplicidade extrema:** fluxo direto, com o mínimo de etapas e decisões.
- **Velocidade:** acesso rápido às oportunidades e execução imediata.
- **Baixa fricção:** pouca burocracia e interface objetiva.
- **Foco em geração de renda rápida:** prioridade para trabalhos com retorno financeiro no curto prazo.
