INSTRUÇÕES PARA TRIGGER DE DEPLOY
==================================

1. Acesse este link:
   https://github.com/GerenciaDriver/bico-brasil/edit/main/README.md

2. Role até o final do arquivo README.md

3. Adicione esta linha no final:
   <!-- Deploy Fallback 500 01:32 -->

4. No campo "Commit message", escreva:
   trigger: deploy fallback 500

5. Clique em "Commit changes"

6. Aguarde 2-3 minutos para o Vercel fazer o deploy

7. Teste o cadastro em: https://bicobrasil.com.br/cadastro

MUDANÇAS NESTE DEPLOY:
- Formulário completo com CPF, WhatsApp e Cidade
- Sanitização de dados (apenas números)
- Fallback inteligente para erro 500
- Login automático se usuário foi criado apesar do erro
